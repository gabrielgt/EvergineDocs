    Gets or sets the playback position relative to the beginning of the PendingBuffers queue.

    Declaration
    public TimeSpan QueuePlayPosition { get; set; }
    Property Value
    Type Description
    TimeSpan
    Remarks

    QueuePosition is ranging from to QueueDuration. Values beyond this range are clamped. When the audio source is in Stopped state and the value is higher than the state will change to Paused. The position is relative to the beginning of all the queued buffers for the source, and any queued buffers traversed by a set value will raise the event BufferEnded.
