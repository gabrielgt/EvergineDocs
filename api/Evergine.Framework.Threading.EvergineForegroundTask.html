    Class EvergineForegroundTask

    Provides methods for creating tasks that will be enqueued in the main Evergine thread.

    Inheritance

    Assembly: Evergine.Framework.dll
    Syntax
    public static class EvergineForegroundTask : object
    Remarks

    The tasks will be enqueued using the ForegroundTaskScheduler.
